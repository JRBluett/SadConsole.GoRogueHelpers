Remarks

Each time Update is called, if RunLogicFrame is true, all entities that are not the ControlledGameObject will, if they have any GameFrameProcessor components, have those component's ProcessGameFrame function called. Then, the controlled game object will have its components ProcessGameFrame functions called. Finally, the RunLogicFrame value is set to false, LogicFrameCompleted event is fired.
