CreateRenderer(Rectangle, Font)

Creates a ScrollingConsole that is configured to render this map. The console will use the map's cell data, and all the console's entities will be properly synced. If you have an existing Console that wasn't created via this function that you would like to configure to render the map, use ConfigureAsRenderer(Console).
